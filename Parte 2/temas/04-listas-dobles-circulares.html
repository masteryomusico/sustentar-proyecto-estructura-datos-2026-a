Módulo 04

🔄 Listas Dobles y Circulares

Listas con punteros bidireccionales o con el último nodo apuntando al primero.

Doble enlaceCircularBidireccional

Documentación

MÓDULO 04

Definición y características

  • En una lista doblemente enlazada cada nodo tiene punteros al nodo anterior y al siguiente.
  • Permite recorrer la lista en ambas direcciones, útil para navegadores e historial de acciones.
  • Una lista circular conecta el último nodo de vuelta al primero, formando un ciclo.
  • Las listas circulares son ideales para turnos rotatorios, playlists o planificación de procesos.

Ejemplo 01

Nodo doble

class NodoDoble:
    def __init__(self, dato):
        self.dato = dato
        self.anterior = None
        self.siguiente = None
Ahora cada nodo conoce a su predecesor y sucesor, permitiendo navegación bidireccional.
Ejemplo 02

Lista circular

n1 = Nodo(1)
n2 = Nodo(2)
n3 = Nodo(3)
n1.siguiente = n2
n2.siguiente = n3
n3.siguiente = n1  # circular!
El último nodo apunta al primero, creando el ciclo. Al recorrer, debemos detectar cuándo volvemos al inicio.
