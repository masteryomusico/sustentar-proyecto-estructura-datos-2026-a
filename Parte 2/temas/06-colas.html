🚶 Colas (Queue)

Estructura FIFO: el primer elemento en entrar es el primero en salir.

MÓDULO 06

Definición y características

  • Una cola (queue) sigue el principio FIFO: First In, First Out.
  • Se usa deque de collections en Python para eficiencia; append() encola y popleft() desencola.
  • Aplicaciones: atención al cliente, cola de impresión, procesamiento de tareas, BFS en grafos.
  • A diferencia de las pilas, la extracción se hace por el frente, no por el fondo.

Ejemplos

2 EJEMPLOS
Ejemplo 01

Encolar clientes

from collections import deque
cola = deque()
cola.append("Cliente 1")
cola.append("Cliente 2")
cola.append("Cliente 3")
Usamos deque para mayor eficiencia. append() añade al final de la cola.
Ejemplo 02

Desencolar

atendido = cola.popleft()
print(atendido)  # Cliente 1
print(cola)      # deque(['Cliente 2', 'Cliente 3'])
popleft() extrae el primer elemento (quien llegó primero). Simula la atención en orden de llegada.
