MÓDULO 08

Definición y características

  • Un árbol binario es una estructura donde cada nodo tiene como máximo dos hijos: izquierdo y derecho.
  • El nodo inicial se llama raíz; los nodos sin hijos son hojas.
  • Los recorridos clásicos son: preorden (raíz, izq, der), inorden (izq, raíz, der) y postorden (izq, der, raíz).
  • Son la base de estructuras más complejas como BST, AVL, heaps y tries.

Ejemplo 01

Clase Nodo árbol

class Nodo:
    def __init__(self, valor):
        self.valor = valor
        self.izq = None
        self.der = None
La clase base del árbol. Cada nodo guarda un valor y referencias a sus dos posibles hijos.
Ejemplo 02

Construir árbol

raiz = Nodo(1)
raiz.izq = Nodo(2)
raiz.der = Nodo(3)
raiz.izq.izq = Nodo(4)
Construimos un árbol manualmente. La raíz es 1, tiene hijos 2 y 3; el nodo 2 tiene hijo izquierdo 4.
