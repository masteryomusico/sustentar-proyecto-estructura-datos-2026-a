Módulo 10

⚖️ Árboles AVL

BST autobalanceado que garantiza O(log n) mediante rotaciones.

BalanceoRotacionesO(log n)

Documentación

MÓDULO 10

Definición y características

  • Un árbol AVL es un BST que se autobalancea: la diferencia de alturas entre subárboles de cualquier nodo es máximo 1.
  • Cuando se viola este balance factor, se aplican rotaciones (simples o dobles) para corregirlo.
  • Existen 4 tipos de rotaciones: derecha, izquierda, izquierda-derecha y derecha-izquierda.
  • Garantizan O(log n) para búsqueda, inserción y eliminación en el peor caso, a diferencia del BST simple.

Ejemplo 01

Rotación derecha

def rotacion_derecha(y):
    x = y.izq
    T2 = x.der
    x.der = y
    y.izq = T2
    y.altura = 1 + max(altura(y.izq), altura(y.der))
    x.altura = 1 + max(altura(x.izq), altura(x.der))
    return x
La rotación derecha se aplica cuando el subárbol izquierdo está más pesado. Reorganiza los nodos manteniendo la propiedad BST.
Ejemplo 02

Factor de balance

def balance(nodo):
    if nodo is None:
        return 0
    return altura(nodo.izq) - altura(nodo.der)

# Si balance > 1: rotación derecha
# Si balance < -1: rotación izquierda
El factor de balance determina qué tipo de rotación se necesita. Un valor fuera de [-1, 1] indica desequilibrio.
