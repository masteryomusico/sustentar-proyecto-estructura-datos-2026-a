# 🌐 Mi Primera Web — Spring Boot + MySQL + HTML + Axios

## 📋 Descripción del Proyecto

Este proyecto consiste en el desarrollo de una aplicación web completa construida con **Spring Boot**, **MySQL**, **HTML** y **Axios**, como parte de un proceso de aprendizaje práctico. El objetivo principal fue crear una primera experiencia real de desarrollo web full-stack, complementada con un manual técnico detallado de todos los temas abordados.

## 📁 Estructura del Proyecto

El proyecto se divide en dos partes:

### Parte 1 — Aplicación Web
Desarrollo de la primera aplicación web integrando el stack tecnológico mencionado.

### Parte 2 — Manual Técnico (Formato ABP)

El manual fue diligenciado según el formato ABP disponible en la plataforma e incluye las siguientes secciones:

- 📄 **Pantalla de Documentación** — Descripción técnica de cada tema abordado.
- 💡 **Pantalla de Ejemplos** — 24 ejemplos prácticos ilustrativos.
- ⚠️ **Pantalla de Retos** — 24 retos identificados *(pendiente de solucionar con recomendaciones)*.
- 🎬 **Clip Explicativo** — Video de autoría del equipo, realizado con herramienta de libre elección.
