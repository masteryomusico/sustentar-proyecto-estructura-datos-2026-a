MÓDULO 01

Definición y características

  • Una estructura de datos es una forma de organizar y almacenar datos en memoria para acceder y modificarlos eficientemente.
  • Son fundamentales porque determinan la eficiencia de los algoritmos que las usan.
  • Permiten organizar información de manera óptima según el problema a resolver.
  • Se clasifican en lineales (arrays, listas, pilas, colas) y no lineales (árboles, grafos).

Ejemplos

2 EJEMPLOS
Ejemplo 01

Lista básica en Python

numeros = [1, 2, 3]
print(numeros)
Creamos una lista con tres enteros y la mostramos en consola. Las listas son la estructura más simple y versátil.
Ejemplo 02

Diccionario clave-valor

agenda = {
  "Ana": "300111"
}
print(agenda["Ana"])
Un diccionario almacena pares clave-valor. Es ideal para búsquedas rápidas por nombre o identificador.
