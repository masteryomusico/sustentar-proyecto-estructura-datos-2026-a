#️⃣ Tablas Hash

Estructura que mapea claves a valores mediante una función hash para acceso O(1).

Documentación

MÓDULO 07

Definición y características

  • Una tabla hash (o diccionario) mapea claves únicas a valores usando una función hash.
  • El acceso, inserción y eliminación son O(1) en promedio, haciéndolas muy eficientes.
  • Las colisiones ocurren cuando dos claves producen el mismo hash; se resuelven con encadenamiento o sondeo.
  • En Python, los diccionarios son tablas hash implementadas de forma nativa.

Ejemplos

2 EJEMPLOS
Ejemplo 01

Crear diccionario

usuario = {
    "nombre": "Juan",
    "edad": 20,
    "email": "juan@email.com"
}
Cada par clave:valor es una entrada en la tabla hash. Las claves deben ser únicas e inmutables.
Ejemplo 02

Acceder y modificar

print(usuario["nombre"])  # Juan
usuario["edad"] = 21
usuario["carrera"] = "Sistemas"
print(usuario)
El acceso por clave es O(1). Podemos actualizar valores existentes o agregar nuevas entradas fácilmente.
