Módulo 09

🔍 BST – Árbol de Búsqueda

Árbol binario donde los valores menores van a la izquierda y mayores a la derecha.

BúsquedaO(log n)Inorden

Documentación

MÓDULO 09

Definición y características

  • En un BST (Binary Search Tree), para cada nodo: todos los valores de su subárbol izquierdo son menores y los del derecho son mayores.
  • Esto permite búsquedas eficientes en O(log n) para árboles balanceados.
  • La inserción mantiene la propiedad BST: comparamos con cada nodo para decidir la dirección.
  • El recorrido inorden de un BST produce los valores en orden ascendente.

Ejemplo 01

Insertar en BST

def insertar(raiz, valor):
    if raiz is None:
        return Nodo(valor)
    if valor < raiz.valor:
        raiz.izq = insertar(raiz.izq, valor)
    else:
        raiz.der = insertar(raiz.der, valor)
    return raiz
La función compara el valor con el nodo actual y recursa hacia el subárbol correspondiente hasta encontrar una posición vacía.
Ejemplo 02

Buscar en BST

def buscar(raiz, objetivo):
    if raiz is None or raiz.valor == objetivo:
        return raiz
    if objetivo < raiz.valor:
        return buscar(raiz.izq, objetivo)
    return buscar(raiz.der, objetivo)
En cada nivel descartamos la mitad del árbol. En un árbol balanceado esto es O(log n).
